# Obrazky jak cyp
A mod that lets you add custom paintings just by drag and dropping!

# Mod Usage
Drag & drop all images you want to see in game, into a folder located at "LethalCompany/BepInEx/Paintings", if it doesn't exist create it.
If you have images that are applied on a texture, rename it to have a '_tex' suffix, example: '<filename>_tex'.
Known limits: Some file types are not supported like webp.
